Arti JavaScript

JavaScript dibuat selama sepuluh hari oleh Brandan Eich, seorang karyawan Perusahaan Netscape, pada bulan September 1995. Awalnya bahasa pemrograman ini dinamakan Mocha, kemudian diganti menjadi Mona, lalu LiveScript dan pada akhirnya resmi menyandang nama JavaScript. Versi awal dari bahasa ini cuma terbatas di kalangan Netscape saja. Fungsionalitas yang ditawarkan pun masih terbatas. Namun, JavaScript terus menerus dikembangkan oleh komunitas developer yang tak pernah berhenti mengerjakan bahasa pemrograman ini.

Arti BootStrap

Bootstrap adalah framework open-source khusus front end yang awalnya dibuat oleh Mark Otto dan Jacob Thornton untuk mempermudah dan mempercepat pengembangan web di front end.
